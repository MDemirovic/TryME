# Examigo Brand Reference

Standalone static reference folder for the Examigo visual system.

Contents:

- `index.html` - brand landing/reference board
- `styles.css` - standalone styles and design tokens in CSS variables
- `tokens.json` - structured palette, typography, and component rules

How to use:

1. Open `index.html` directly in a browser.
2. Use the page as the visual reference for palette, typography, buttons, fields, radio cards, badges, and mascot direction.
3. Use `tokens.json` as the source of truth when translating these decisions into Figma or future app components.

This folder is intentionally separate from the Expo mobile app and is not imported anywhere in the app code.
