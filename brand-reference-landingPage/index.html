Brand summary

What Examigo should always feel like

  • Warm and premium, not sterile.
  • Smart examiner energy, not generic AI assistant.
  • Document-first studying, not dashboard-first clutter.
Mascot rules

Exa usage direction

  • Rounded silhouette, expressive eyes, slightly serious attitude.
  • Feels intelligent before cute.
  • Appears in hero moments, loading, empty states, and gentle feedback.
Voice direction

Interface tone

  • Supportive, clean, and a little sharp.
  • Never childish, never corporate.
  • Short copy with confidence and clarity.

Typography

Google Fonts

Sora carries the brand personality. Manrope keeps everything readable and calm.

Heading font

Sora

Examigo gets you ready.

Use 700 or 800 for hero headlines and section anchors.

Body font

Manrope

Built for paragraphs, labels, helper text, input content, and navigation.

Use 400 for default copy and 600 for support emphasis.

Buttons

Button system

Buttons should feel roomy, friendly, and obvious. Default radius stays pill-shaped.

Primary full width

Height 58-62px. Use for primary flows and single obvious actions.

Secondary full width

Use on mint-wash background with dark text for calmer secondary actions.

Ghost / text action

Use for dismissive or less important choices. Keep it quiet but still readable.

Icon + label row

Good for authentication or multi-option utility rows.

Reference rules

Keep using these patterns

Cards

Rounded 22-30px, off-white fill, soft border, subtle warm shadow.

Spacing

Generous vertical rhythm. Let hero, cards, and sections breathe.

Icons

Outline icons by default. Use teal for active and ink for neutral.

Mascot placement

Hero moments, onboarding wow states, empty states, completion moments.
